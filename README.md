# Python-Pandas

🔹 Project Description

This repository contains a set of 5 Pandas mini projects designed to build strong foundations in data manipulation and analysis using Python.
Each project focuses on applying Pandas functionalities such as data cleaning, filtering, grouping, merging, aggregation, and visualization to solve practical data analysis problems.
These projects help in understanding how to handle structured data efficiently and extract meaningful insights.

🔹 Project Objectives

Learn and apply core Pandas operations
Perform data cleaning and preprocessing
Work with DataFrames and Series
Apply filtering, sorting, and grouping techniques
Perform exploratory data analysis (EDA)
Generate insights from datasets

🔹 Project-wise Description

✅ Project 1: Data Loading & Basic Exploration
Reading CSV/Excel files using Pandas
Displaying data using head(), tail(), info()
Understanding dataset structure

✅ Project 2: Data Cleaning
Handling missing values (isnull(), dropna(), fillna())
Removing duplicates
Fixing incorrect or inconsistent data

✅ Project 3: Filtering & Sorting Data
Applying conditions to filter rows
Using boolean indexing
Sorting data using sort_values()

✅ Project 4: Grouping & Aggregation
Using groupby() for grouped analysis
Applying aggregation functions like sum(), mean(), count()
Extracting insights from grouped data

✅ Project 5: Data Analysis & Visualization
Performing exploratory data analysis (EDA)
Using Pandas plotting functions
Creating:
Line plots
Scatter plots
Histograms
Drawing insights from visual data
